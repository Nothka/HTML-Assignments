Simple Multi‑Page Website (HTML only, no CSS)

Files:
- index.html         → homepage (with a link to the list page)
- books.html         → list of items (each item links to its own page)
- book-dune.html     → detail page (text + image)
- book-hp1.html      → detail page (text + image)
- book-1984.html     → detail page (text + image)
- images/placeholder.png → placeholder image you can replace

How to use:
1) Open index.html in your browser to start.
2) Replace images/placeholder.png with your own images, or change the <img src="..."> paths.
3) Edit the text in each file to match your own hobby, favorite movies, games, etc.
4) Submit the folder or a ZIP of it as required.

RO:
1) Deschide index.html în browser.
2) Înlocuiește images/placeholder.png cu imaginile tale, sau schimbă calea în atributul src.
3) Editează textul din fișiere pentru hobby-ul tău / filme / jocuri etc.
4) Predă folderul sau arhiva ZIP, conform cerințelor.
